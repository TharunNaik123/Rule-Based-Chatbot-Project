# Rule-Based Chatbot (Python)


A simple **console-based rule chatbot** built using Python. It supports:
- Pattern matching
- Small talk
- Help and greeting intents
- A small knowledge base
- Conversation logging


---


Features
- Pattern-matching based intent recognition
- Domain question answering
- Logs all conversations
- Lightweight and easy to run
